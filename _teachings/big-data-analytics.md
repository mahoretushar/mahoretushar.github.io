## Course Overview

This course covers frameworks and techniques for working with data that exceeds the capacity of traditional tools — from distributed storage and processing to real-time stream analytics.

## Topics Covered

- Introduction to Big Data — 5 Vs, ecosystem overview
- Hadoop ecosystem: HDFS, MapReduce, YARN
- Apache Spark: RDDs, DataFrames, Spark SQL
- NoSQL databases: HBase, Cassandra, MongoDB
- Stream processing: Kafka, Spark Streaming
- Cloud platforms: Google BigQuery, AWS EMR, Databricks
- Big Data visualisation and dashboarding

## Tools

Hadoop, Spark, Hive, Kafka, BigQuery, Databricks, Power BI
