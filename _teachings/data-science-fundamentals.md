## Course Overview

This course equips students with end-to-end data science skills — from raw data to actionable insights. Emphasis is on building intuition alongside practical implementation skills.

Students completing this course will be able to:

- Apply statistical reasoning to real-world datasets
- Clean, transform, and explore data using Python (Pandas, NumPy)
- Build and evaluate machine learning models with Scikit-learn
- Create compelling visualizations using Matplotlib, Seaborn, and Tableau
- Present data-driven findings clearly and accurately

## Tools & Environment

- **Languages:** Python, R
- **Libraries:** Pandas, NumPy, Matplotlib, Seaborn, Scikit-learn
- **Tools:** Jupyter Notebooks, Anaconda, Power BI, Tableau, BigQuery

## Prerequisites

- Basic programming knowledge (Python preferred)
- High school mathematics (algebra, basic statistics)
