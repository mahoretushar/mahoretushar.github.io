## Course Overview

Covers the theoretical foundations and practical techniques for protecting networked systems — from cryptographic primitives to modern threat detection.

## Topics Covered

- Security fundamentals: CIA triad, threats, vulnerabilities, attacks
- Symmetric cryptography: DES, AES, modes of operation
- Asymmetric cryptography: RSA, Diffie-Hellman, ECC
- Hash functions: SHA family, MD5, HMAC
- Digital signatures and PKI
- Authentication protocols: Kerberos, TLS/SSL handshake
- Firewalls: packet filtering, stateful, application-layer
- Intrusion Detection and Prevention Systems (IDS/IPS)
- VPNs: IPSec, SSL VPN
- Common attacks: SQL injection, XSS, MITM, phishing, DoS/DDoS
- Wireless network security: WPA2/3, rogue APs
- Ethical hacking and penetration testing basics

## Tools

Wireshark, Kali Linux, Nmap, OpenSSL
