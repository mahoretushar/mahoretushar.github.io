## Course Overview

An introductory programming course using C — one of the most enduring systems programming languages and the foundation for understanding how computers execute code.

## Topics Covered

- Introduction to programming: flowcharts, algorithms, pseudocode
- C basics: data types, variables, operators, I/O
- Control structures: if-else, switch, loops (for, while, do-while)
- Functions: definition, arguments, return types, recursion
- Arrays: 1D, 2D, string handling
- Pointers: pointer arithmetic, pointers and arrays, dynamic memory (malloc/free)
- Structures and unions
- File I/O in C
- Introduction to preprocessor directives
