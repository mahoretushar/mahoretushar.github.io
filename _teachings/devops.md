## Course Overview

A practical course equipping students with the DevOps skillset demanded by modern software teams — from source control to automated deployment.

## Topics Covered

- Linux command line: file system, permissions, shell scripting
- Version control with Git and GitHub: branching, merging, pull requests
- Continuous Integration: GitHub Actions, Jenkins basics
- Containerisation: Docker — images, containers, volumes, networking
- Docker Compose for multi-container applications
- Introduction to Kubernetes concepts
- Configuration management basics: Ansible
- Monitoring and logging fundamentals
- Cloud deployment: AWS / Azure basics

## Tools

Git, GitHub, Docker, Jenkins, Linux (Ubuntu/RedHat), Ansible
