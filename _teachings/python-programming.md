## Course Overview

A practical, hands-on Python programming course designed to take students from zero to building real applications. Emphasis on clean code, problem-solving, and industry-relevant workflows.

## Topics Covered

- Python basics: syntax, variables, data types, operators
- Control flow: conditionals, loops, comprehensions
- Functions, lambda expressions, decorators
- Data structures: lists, tuples, sets, dictionaries
- Object-Oriented Programming: classes, inheritance, polymorphism
- File I/O and exception handling
- Modules and packages
- Regular expressions
- Working with APIs and JSON
- Introduction to NumPy and Pandas for data tasks

## Tools

Python 3, Jupyter Notebooks, VS Code, Anaconda
