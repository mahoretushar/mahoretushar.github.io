## Course Overview

Introduces students to object-oriented programming using Java — one of the most widely used languages in enterprise and Android development.

## Topics Covered

- Java fundamentals: JVM, JDK, JRE, syntax basics
- OOP concepts: classes, objects, inheritance, polymorphism, abstraction, encapsulation
- Interfaces and abstract classes
- Exception handling: try/catch/finally, custom exceptions
- Collections framework: ArrayList, HashMap, LinkedList, Iterator
- Generics
- File I/O and serialisation
- Multithreading and synchronisation
- Introduction to Java Swing (GUI basics)
- JDBC and database connectivity

## Tools

Java SE, Eclipse / IntelliJ IDEA, NetBeans
