## Course Overview

Builds on core DBMS concepts to cover distributed and parallel databases, data warehousing, OLAP, and modern large-scale database architectures.

## Topics Covered

- Distributed database systems: design, fragmentation, replication
- Distributed query processing and optimisation
- Data warehousing: star schema, snowflake schema, ETL
- OLAP and multidimensional data models
- Spatial and temporal databases
- XML databases and semi-structured data
- NewSQL and cloud-native databases
- Database security and authorisation

## Tools

Oracle, PostgreSQL, MySQL, MongoDB, Hive
