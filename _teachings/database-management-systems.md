## Course Overview

A foundational course in database design and management — from ER modelling to SQL programming to transaction control and storage internals.

## Topics Covered

- Data models: hierarchical, network, relational
- Entity-Relationship (ER) modelling and ER-to-relational mapping
- Relational algebra and SQL (DDL, DML, DCL, TCL)
- Normalisation: 1NF through BCNF
- Transactions: ACID properties, concurrency control, locking, deadlock
- Recovery techniques
- Query optimisation basics
- Introduction to NoSQL: document, key-value, column, graph stores

## Tools

MySQL, Oracle, PostgreSQL, MongoDB
