## Course Overview

Data mining bridges database systems and machine learning — extracting actionable knowledge from raw data. This course covers the core algorithmic toolkit and practical workflows.

## Topics Covered

- KDD process and data preprocessing
- Classification: Decision Trees, Naïve Bayes, k-NN, SVM
- Clustering: k-Means, hierarchical, DBSCAN
- Association rule mining: Apriori, FP-Growth
- Anomaly / outlier detection
- Regression and prediction
- Text mining and web mining basics
- Evaluation metrics for mining tasks

## Tools

Python (Scikit-learn, NLTK), Weka, KNIME
